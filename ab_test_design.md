# A/B Test: paymentChange

## Case Description

The product is a mobile application operating under a freemium model. Users can access basic functionality for free, while full access is available through a paid subscription.

The product monetizes through:
- subscription (main revenue stream)
- one-time purchases

A pricing experiment was conducted in which the subscription cost was reduced from 400 to 160 for new users using a 50/50 split.

---

## Objective

The goal of this project is to perform a full A/B test analysis cycle, including:

- defining business requirements  
- designing the experiment  
- validating experimental setup  
- analyzing results  
- providing product recommendations  

---

## Project Structure

- `README.md` — case description and overview  
- `docs/ab_test_design.md` — full experiment design  
- `notebooks/ab_test_analysis.ipynb` — data analysis and statistical testing
